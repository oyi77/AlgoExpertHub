# Cursor Commands

This directory contains custom Cursor IDE commands for Spec-Driven Development (SDD).

## Available Commands

### Quick Mode (80% of features)
- **`/brief`** - Create a lightweight 30-minute feature brief

### Full SDD 2.0 Workflow (20% of features)
- **`/research`** - Research technologies and best practices
- **`/specify`** - Create comprehensive feature specification
- **`/plan`** - Design technical architecture and implementation plan
- **`/tasks`** - Break down feature into actionable tasks
- **`/implement`** - Implement feature following specs and plan

### Management Commands
- **`/evolve`** - Update specifications as development progresses
- **`/upgrade`** - Convert feature brief to full SDD workflow

## Usage

Type `/` followed by the command name in Cursor's chat to use these commands.

Example:
```
/brief user-authentication Add OAuth2 authentication with Google and GitHub
```
