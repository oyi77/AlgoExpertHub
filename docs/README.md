## üìñ Documentation Structure

All documentation is organized in this `docs/` folder and linked from the main README. When creating new documentation:

1. Create document in `docs/` folder
2. Add link to this README
3. Add reference in main README.md if needed
4. Use Mermaid diagrams for flow charts

### Documentation Standards

- **Format**: Markdown (.md)
- **Diagrams**: Use Mermaid syntax for flowcharts and diagrams
- **Code Examples**: Include working code snippets
- **Structure**: Clear sections with headings
- **Links**: Use relative links within docs folder
